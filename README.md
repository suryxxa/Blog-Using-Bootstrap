# Blog-Using-Bootstrap
This is a simple social post blog created using Bootstrap to ensure a fully responsive design across all devices. The project includes 6 blog posts and a contact form, all contained within a single index.html file. The main goal of this project was to learn and practice Bootstrap and improve responsive design skills.
